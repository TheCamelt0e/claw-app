## The Solution: Context-Aware Resurfacing

### Core Innovation
CLAW doesn't just store intentionsâ€”it **understands context** and surfaces them when:
- You're near a relevant location
- You open a specific app (Amazon, Netflix)
- Time context matches (evening, weekend)
- Activity context aligns (driving, shopping)

### The Hook
> "I got a notification for a book while browsing Amazon. I bought it instantly. I've been trying to remember that book for 3 weeks."

### Key Differentiators
1. **AI-Native:** LLM-powered categorization & context extraction
2. **Frictionless:** Voice-first, 3-second capture
3. **Smart Expiration:** 7-day default prevents hoarding
4. **Social:** "Claw Forward" viral mechanic
